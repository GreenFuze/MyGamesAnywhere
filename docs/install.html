Windows portable preview

Install MGA without an installer.

MGA v0.0.5 ships as a Windows portable ZIP. It runs locally, stores data beside the runtime, and binds to 127.0.0.1 by default.

  1. Download mga-v0.0.5-windows-amd64-portable.zip from GitHub Releases.
  2. Extract it into a writable folder such as C:\Games\MGA.
  3. Run Start MGA.cmd.
  4. Open http://127.0.0.1:8080.

Avoid extracting the portable build under Program Files. The current runtime writes config, database, media, and local state inside the portable folder.

Upgrade policy

  1. Stop MGA.
  2. Back up config.json, data/, and media/.
  3. Replace the bundled binaries, plugins, and frontend files with the new package.
  4. Start MGA and verify the version in About.
