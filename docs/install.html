Windows portable release

Install MGA without an installer.

MGA v0.0.7 ships as a Windows portable ZIP. It runs locally, stores runtime state beside the portable folder, and binds to 127.0.0.1:8900 by default.

  1. Download mga-v0.0.7-windows-amd64-portable.zip from GitHub Releases.
  2. Extract it into a writable folder such as C:\Games\MGA.
  3. Run Start MGA.cmd.
  4. Open http://127.0.0.1:8900.

Local-first runtime

The current package behaves more like a self-hosted game library running on your own machine than a cloud-first account shell.

  • local server
  • local database
  • local media cache
  • local config
  • plugin-backed integrations

Portable build notes

Avoid extracting the portable build under Program Files. The current runtime writes config, database, media, and local state inside the portable folder. The shipped config uses LISTEN_IP: "127.0.0.1" and PORT: "8900"; LAN binding is opt-in by editing config.json.

Upgrade policy

  1. Stop MGA.
  2. Back up config.json, data/, and media/.
  3. Replace the bundled binaries, plugins, and frontend files with the new package.
  4. Start MGA and verify the version in About.
