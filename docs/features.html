What MGA does

Local-first game launcher for multi-source collections.

Canonical merge

MGA merges source entries into one canonical game when detection can prove the match.

Source provenance

Each game keeps visible source records, files, runtime flags, and provider metadata.

Manual review

Undetected games stay visible and can be searched, matched, or re-detected later.

Achievements

Cached Steam, Xbox, and RetroAchievements progress is available in dashboard and per-game views.

Browser play

Configured browser runtimes such as EmulatorJS, js-dos, and ScummVM can launch from the web UI.

Sync surfaces

Save-sync and settings-sync integrations are part of the plugin model.
