
# UI/UX

## Screens (MVP)
1. Library
   - grid/list toggle
   - search
   - filters: source, installed
   - tag/category filters
2. Game Details
   - launch button
   - tags/categories editor
   - installer association panel (for Drive installer entries)
   - metadata + artwork display
   - “Refresh metadata/media” action
3. Scan / Import
   - Steam scan
   - Drive folders manager (add/remove folder IDs)
   - Drive scan results list with match workflow
4. Plugins
   - list plugins, enable/disable
   - install plugin from zip
   - configure plugin settings (render from JSON schema)
   - view last error + diagnostics
5. Sync
   - connect/disconnect Drive
   - last sync time + manual sync
6. Settings
   - data/cache locations
   - export diagnostics bundle

## Key flows
- Drive folder selection → scan → match candidates → confirm mapping
- Library → details → launch → playtime recorded
- Plugins → install zip → enable → configure keys
