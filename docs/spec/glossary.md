
# Glossary

- **Canonical (cloud)**: The JSON files on Drive used for sync across devices.
- **Local cache**: SQLite + filesystem cache on the device.
- **Source plugin**: Provides scan/launch for a game source (Steam, Drive installers).
- **Provider plugin**: Provides metadata or media fetching (IGDB, SteamGridDB, LaunchBox export).
- **InstallerRef**: Reference to a setup file on Drive (by fileId).
- **LWW**: Last-write-wins merge using timestamps.
