# MyGamesAnywhere v0.0.6

## Highlights

- Xbox achievement unlock-state parsing now reflects actual unlocked counts instead of treating locked entries as achieved
- YouTube video media now shows thumbnails, inline preview support, and correct original links
- featured media rails now include a previewable video when a game has video media available
- game-page achievement summaries now use a progress ring and denser card layout
- README and GitHub Pages now position MGA around canonical game identity, provenance, and local-first ownership

## Packaging Notes

- artifact name: `mga-v0.0.6-windows-amd64-portable.zip`
- MGA remains a Windows portable working-directory-based runtime in this release
- the public release flow now uses local packaging plus manual `gh` release publication instead of tag-triggered auto-publishing

## Upgrade Notes

- this release keeps the same portable replacement-style upgrade path as `v0.0.5`
- back up `config.json`, `data/`, and `media/` before replacing an existing runtime
- verify the About page version/build metadata after upgrading
