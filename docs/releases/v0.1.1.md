# v0.1.1 Release Notes

`v0.1.1` is a metadata reliability and desktop polish release after `v0.1.0`.

## Highlights

- Metadata refresh no longer aborts when a provider (e.g. RAWG, Steam) times out or returns an error. The refresh continues with all remaining providers and surfaces a warning on the game detail page showing which provider was skipped and why.
- The desktop app no longer opens a terminal window when launched via the build script — it now starts silently as a true Windows GUI process.
- The application icon is now correctly embedded in the desktop `.exe`, showing the MGA icon in Explorer and the taskbar.
- The desktop client now automatically clears the active profile when switching servers, preventing stale profile state across connections.

## Upgrade Notes

- `NO_MIGRATION_NEEDED`: v0.1.1 fixes server metadata refresh behavior and desktop launch/icon issues. No SQLite schema, persisted config, or sync payload changes are introduced.
- Existing installs can upgrade normally through the installer or portable update flow.
