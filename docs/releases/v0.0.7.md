# MyGamesAnywhere v0.0.7

## Highlights

- safer file-backed delete flows for Game Detail source records and Undetected Games candidates
- plugin-backed dry-delete previews show the exact files that will be affected before the real delete action is available
- Google Drive source deletes move explicit files to Drive trash; SMB deletes only explicit file targets
- destructive file delete dialogs now require a checkbox confirmation before the final delete button is enabled
- server `LISTEN_IP` config supports local-only and opt-in LAN binding while portable releases ship with `127.0.0.1:8900`
- manual metadata search is broader, fuzzy, platform-aware, and provider lookups run concurrently
- long manual-search descriptions are truncated by default and expand/collapse on click

## Packaging

- artifact name: `mga-v0.0.7-windows-amd64-portable.zip`
- released portable config includes `PORT: "8900"` and `LISTEN_IP: "127.0.0.1"`
- MGA remains a Windows portable working-directory-based runtime in this release
- no installer, service mode, firewall automation, or Settings-host editor is included yet

## Upgrade Notes

- this release keeps the same portable replacement-style upgrade path as the previous release
- stop MGA before replacing files
- back up `config.json`, `data/`, and `media/` before upgrading
- if you previously copied an older `config.json`, make sure it includes `LISTEN_IP`
- LAN exposure remains opt-in by editing `config.json`; released ZIPs stay local-only by default
