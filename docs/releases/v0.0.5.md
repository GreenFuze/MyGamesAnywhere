# MyGamesAnywhere v0.0.5

## Highlights

- first portable Windows packaging track for MGA
- local-only packaged runtime on `127.0.0.1:8080`
- versioned release flow backed by `VERSION` and Git tags
- README and release policy now reflect the real packaging and upgrade story

## Packaging Notes

- artifact name: `mga-v0.0.5-windows-amd64-portable.zip`
- MGA remains a portable working-directory-based runtime in this release
- no installer, service mode, or LAN-binding flow is included yet

## Upgrade Notes

- this release uses a portable replacement-style upgrade path
- back up `config.json`, `data/`, and `media/` before replacing an existing runtime
- verify the About page version/build metadata after upgrading
