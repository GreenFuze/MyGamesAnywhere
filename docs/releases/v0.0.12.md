# v0.0.12 Release Notes

`v0.0.12` is a web-client polish and public-docs release after `v0.0.11`.

## Highlights

- Refreshed README and GitHub Pages as a marketing + quick-install surface for MGA.
- Added current v0.0.12 screenshots for Play, Achievements, Library Statistics, Duplicates, and Undetected Games.
- Undetected Games can now manually mark a source record as DLC/Add-on content and restore it as a base game.
- Manual add-on decisions update existing source-game `kind` and `review_state` rows, then recompute canonical grouping.
- Library now includes a detailed List view with row selection, conservative bulk reclassify queueing, and shared two-stage hard-delete preview/apply behavior.
- GitHub Pages now calls out the local web client, library statistics, add-on review controls, canonical split/merge, duplicate cleanup, and provider-aware achievements more clearly.

## Upgrade Notes

- `NO_MIGRATION_NEEDED`: v0.0.12 manual DLC/Add-on controls update existing `source_games.kind` and `source_games.review_state` values only. No SQLite schema, persisted config, or sync payload changes are introduced.
- `NO_MIGRATION_NEEDED`: the Library List view uses the existing persisted frontend `viewMode` contract value and existing source hard-delete/reclassify APIs; no new schema or config shape is introduced.
- Existing installs can upgrade normally through the installer or portable update flow.
