# v0.0.11 Release Notes

`v0.0.11` is a focused Windows portable and installer fix release after `v0.0.10`.

## Highlights

- Statistics ranked bars now deep-link into Library filters, so platform/source breakdowns can be clicked directly into filtered library views.
- Steam achievement refresh treats apps whose schema endpoint returns `400 {}` as games with no achievements instead of surfacing a provider error.
- Packed Windows installer add-ons with strong signals such as `level_pack`, `character_pack`, `movie_level_pack`, and `season_pass` are archived as add-on content instead of staying in the active Undetected Games queue.
- Undetected Games candidate titles now wrap long installer filenames inside their cards and detail headers.

## Upgrade Notes

- No DB migration is required for this release.
- `NO_MIGRATION_NEEDED`: the add-on installer change uses existing source-game `kind` and `review_state` fields; existing installs remain safe.
- Existing add-on installer rows may need a metadata refresh or rescan to be reclassified.
